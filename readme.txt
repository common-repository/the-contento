== Description ==

#### Contento for WordPress

*Searching and publishing contents for your wordpress plugin should be easy. With this plugin, it finally is.*

Contento for WordPress helps you add find and publish contents for your wordpress sites.
#### Some of the Contento for WordPress features

- Connect with your Contento account in seconds.
- Free 10 days trial for selected relevant contents
- Select default description that will be used to generate automatic contents on your wordpress site.
- Find a minimum of 200 relevant contents on the contento plugin dashboard on your wordpress admin area.
- View, Add to Draft and Publish options for all contents generated.
- Automatic upload of images to your wordpress server for all contents (No more manually downloading of images).
- Saves you time, resources and efforts.
- It also have a functionality to help you autopublish contents based on keywords and auto-publishing settings you've enabled.

#### What is Contento?

Contento is a subscription based content providing service that provides automatic contents for your site leveraging on a unique algorithm developed by Femtosh Global Solutions to find already published contents on the internet. Contento is free for 10 days for every domain.

This plugin acts as a bridge between your WordPress site and your Contento account, connecting the two together.

If you do not yet have a Contento account, [creating one is 100% free and only takes you about 30 seconds](http://www.contento.com.ng/register)

== Installation ==

1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **The Contento** and click "*Install now*"
2. Alternatively, download the plugin and upload the contents of `wp-contento.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
3. Activate the plugin
4. Set [your Contento username, password](https://www.contento.com.ng/user) in the plugin settings.
5. Set your default subscription also on the settings page after the username and the password have been updated.
